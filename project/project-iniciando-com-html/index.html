A rede mundial de computadores

A web é composta, em grande parte, por documentos HTML. E a mesma, vem sendo utilizada por milhares de pessoas. Acessada por diversos tipos de dispositivos.

Tipos de arquivos encotrados na WEB

  • Plain text
  • HTML
  • Imagens
  • Vídeos
  • Músicas

Componentes da WEB

Os componentes da web incluem o cliente (seu computador e navegador), a Internet e os servidores. O cliente e os servidores interagem por meio de algo chamado Hypertext Transfer Protocol - HTTP (protocolo de transferência de hipertextos).

Conceitos básicos de HTML

O HTML significa HyperText Markup Language (linguagem de marcação de hipertextos) e é a linguagem que os computadores usam para interpretar páginas web. Os componentes da web incluem o cliente (seu computador e navegador), a Internet e os servidores. O cliente e os servidores interagem por meio de algo chamado Hypertext Transfer Protocol - HTTP (protocolo de transferência de hipertextos).

Exemplo de uma estrutura HTML

Explicando a estrutura

  • A declaração <!DOCTYPE html> define o documento como html.
  • No elemento <html lang="pt"> é definidoo elemento raiz da página "html" e a linguagem "pt".
  • No elemento <head> é definido os metadadosda página.
  • No elemtento <title> é definido o título do documento.
  • O elemtento <body> contém o conteúdo visível da página.
  • No elemento <h1> define o cabeçalho.
  • O elemento <p> define um paragrafo.

Tags vistas no curso

Tag Descrição
<b> é utilizada para deixar o texto em negrito
<em> é utilizada para deixar o texto em italico
<a> utilizando a tag <a> com o atributo href="" é utilizada para definir um hiperlink.
<br/> é utilizada para inserir uma quebra de linha
<p> é utilizada para adicionar um paragrafo
